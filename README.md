
# Eye Care AI Workflow Prototype

This is a simplistic prototype model for an AI-powered eye care clinic workflow system.

## What it does

The model takes patient intake data and produces:

- Triage level
- SOAP-style chart note
- Scheduling recommendation
- Prior authorization flag
- Dashboard-ready patient summary

## Run it

```bash
python eye_care_workflow_model.py
```

## Example Output

The model outputs structured JSON that could be shown in a physician, technician, or RCM dashboard.

## Important Note

This is not a diagnostic medical model. It is a workflow automation prototype designed for human review.
